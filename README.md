# Retail Sales Data Analysis

## 📌 Project Overview
Performed exploratory data analysis on retail sales data to identify profit trends, seasonal patterns, and top-performing regions.

## 🎯 Objectives
- Identify top & bottom performing states
- Analyze sales vs profit relationship
- Detect seasonal trends
- Calculate KPIs (Sales, Profit, Order Count, Profit Ratio)

## 🛠 Tools Used
- Excel
- Python 
- Data Visualization
- Pivot Tables

## 📊 Key Insights
- Certain states show high sales but low profit margins
- Seasonal spikes observed in Q4
- Profit ratio varies significantly by category
